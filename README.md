## Goals

- All original solutions
- No external libraries (Python standard library only)
- Reasonably quick solutions (&lt;10s each)

## Getting Started

    poetry install
    poetry shell
    ...

## Run Tests and Solve Problems

    pytest [-k <problem number>]
